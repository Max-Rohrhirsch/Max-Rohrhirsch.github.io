Description Command
.asm to .bin nasm -f bin boot.asm -o boot.bin
Run .bin qemu-system-x86_64 boot.bin
Both nasm -f bin boot.asm -o boot.bin && qemu-system-x86_64 boot.bin
.cpp to .o i386-elf-gcc –ffreestanding –m32 –g –c "kernel.cpp" -o "kernel.o"
.asm to .o nasm "kernel_entry.asm" -f elf –o "kernel_entry.o"
Link them i386-elf-ld –o "full_kernel.bin" -Ttext 0x1000 "kernel_entry.o" "kernel.o" --oformat binary && nasm "boot.asm" -f bin –o "boot.bin" && Cat "boot.bin" "full_kernel.bin" > "everything.bin"
