Optimizations



Constant Folding

Optimization: Evaluate constant expressions at compile-time.


int result = 2 + 3;



int result = 5;

Dead Code Elimination

Code that is not used is not compiled


int x = 5;
int myFunc()...




// no code

Common Subexpression Elimination


int result = (x + y) * (x + y);



int common = x + y;
int result = common * common;

Inlining

Optimization: Replace function calls with the actual function body.


inline int add(int a, int b) {
    return a + b;
}
int result = add(2, 3);



result = 2 + 3

Strength Reduction

Optimization: Replace expensive operations with equivalent but cheaper ones.


for (int i = 0; i < n; i++) {
    result += 2 * i;
}



for (int i = 0; i < n; i++) {
    result += i << 1;
}

Constant Propagation

Optimization: Replace variables with their constant values when possible.


const int x = 42;
int y = x + 10;



int y = 52;
