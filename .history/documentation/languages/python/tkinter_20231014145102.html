Basics Tkinter (GUI)

Python


With Tkinter you can easily create a window and all the elements. Dont forget to put the main Loop on the end!!!

                # Basics

from tkinter import * 
from tkinter.ttk import * 

root = Tk()
root.geometry('100x100') 
 
# Create a Button
btn = Button(root, text = 'Click me !', bd = '5',
                          command = root.destroy) 
 
# Set the position of button on the top of window.   
btn.pack(side = 'top')    
root.mainloop()
