Tkinter - Python GUI

Installation​

sudo apt-get install python3-tk

Basic Window​

from tkinter import * 
from tkinter.ttk import * 

root = Tk()
root.title('MyTitle') 
root.geometry('100x100')  
 
# Create a Button
btn = Button(root, text = 'Click me !', bd = '5',
                          command = root.destroy) 
 
# Set the position of button on the top of window.   
btn.pack(side = 'top')    
root.mainloop()
